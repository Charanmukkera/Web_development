# 💰 Expense Tracker

The **Expense Tracker** is a simple yet powerful **web application** designed to help users manage their daily expenses efficiently.  
It allows you to **add, view, and manage expenses** while automatically organizing them by **year, month, and category**.  
The project features a **clean, modern design** with a user-friendly interface.

---

## 📌 Features

### ✅ Expense Management
- Add expenses with **category**, **amount**, and **date**.
- Automatically sorts expenses into **separate yearly tables**.
- Calculates **yearly totals** so you can quickly see how much you spent.

### ✅ Smart Date Validation
- Prevents adding expenses for **future years** to maintain accurate records.

### ✅ Professional UI
- Clean, **colorful gradient background** for a modern look.
- Responsive layout for **desktop and laptop** screens.
- Highlighted **total rows** for each year.

### ✅ Organized by Year & Month
- Separate tables for each year.
- Displays month and category for every expense.
